## Installation

1. Open **Extensions** in VS Code (`Cmd+Shift+X` / `Ctrl+Shift+X`)
2. Search for `Funchosa Dark Theme`
3. Click **Install**
4. Press `Cmd+K Cmd+T` / `Ctrl+K Ctrl+T` and select **Funchosa Dark Theme**

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Background | `#1f1d2e` | Editor background |
| Foreground | `#e0def4` | Main text |
| Rose | `#eb6f92` | Tags, errors |
| Peach | `#ebbcba` | Accents, operators |
| Gold | `#f6c177` | Numbers, classes |
| Foam | `#9ccfd8` | Strings, properties |
| Iris | `#c4a7e7` | Keywords |
| Pine | `#31748f` | Functions |

## Recommended Settings

```json
{
  "editor.fontFamily": "'Fira Code', 'JetBrains Mono', monospace",
  "editor.fontLigatures": true,
  "editor.fontSize": 14,
  "editor.lineHeight": 1.6,
  "editor.bracketPairColorization.enabled": true
}
```
